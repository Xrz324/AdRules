## Description

本仓库源自 [Cats-Team维护的AdRules](#https://github.com/Cats-Team/AdRules)
出于个人使用目的进行了修改，并采用了更偏激的规则，如果有帮到你请感谢原开发团队，感谢原项目的付出和开发。

## Rules

- 内容过滤
  - `adblock.txt`
- DNS 过滤（ABP 语法）
  - `dns.txt`
- 代理工具规则集
  - `adrules-singbox.srs`
  - `adrules-mihomo.mrs`
- Mihomo 补充（用于覆盖 mrs 无法表达的规则）
  - `adrules-mihomo.yaml`
